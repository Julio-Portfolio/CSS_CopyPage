# Clone do Site Discord

Este projeto é um clone responsivo do site Discord utilizando apenas HTML e CSS. O layout foi criado com CSS Grid para garantir que o site seja flexível e se adapte a diferentes tamanhos de tela. O design imita de forma próxima a página inicial do Discord, mas é exclusivamente para fins educacionais.

## Funcionalidades

- **Design Responsivo**: O site se ajusta a diferentes tamanhos de tela utilizando media queries, garantindo uma experiência fluida tanto em desktops quanto em dispositivos móveis.
- **Layout com Grid**: O layout foi criado com CSS Grid, proporcionando uma abordagem moderna e estruturada para posicionar os elementos na página.
- **Fontes Customizadas**: O site utiliza as fontes do Google Fonts para proporcionar um design limpo e legível com as fontes 'Roboto' e 'Anton'.
- **Imagens de Fundo**: O site utiliza diferentes imagens de fundo para várias seções, criando uma experiência visualmente envolvente.

## Tecnologias Usadas

- **HTML**: Linguagem de marcação para a estrutura do site.
- **CSS**: Utilizado para estilizar e criar o layout com CSS Grid e Flexbox para a responsividade.
- **Google Fonts**: Para tipografia customizada com as fontes 'Roboto' e 'Anton'.
